## Installation

1. Extract the ZIP archive to any folder (e.g., `C:\Program Files\Silence`)
2. Run `silence!.exe`
3. (Optional) Enable "Start with Windows" in settings

## Usage

- **Hotkey**: Press Ctrl+Alt+M (or your custom hotkey) to toggle mute
- **Tray Icon**: Left-click to mute/unmute, right-click for menu
- **Settings**: Double-click tray icon to open settings window

## System Requirements

- Windows 10 (build 19041) or Windows 11
- x64 processor

## Notes

- No installation required - portable application
- All settings saved locally
- No internet connection required
